Responsive Image Gallery

Description
This project is a responsive image gallery created as part of my Frontend Development Internship at CodeAlpha. It provides an attractive way to display nature images with a clean and user-friendly interface.

Features
- Responsive gallery layout
- Interactive image display
- Modern user interface
- Easy navigation
- Nature-themed gallery

Technologies Used
- HTML5
- CSS3
- JavaScript

How to Run
Download the project files and open `index.html` in any web browser.
